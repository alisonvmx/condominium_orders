# Quasar App (condominium-orders)

A Quasar Project

## Install the dependencies
```bash
yarn
# or
npm install
```

### inicar o projeto
```bash
yarn dev


```
### inicar a api
```bash
yarn startApi


```


### Lint the files
```bash
yarn lint
# or
npm run lint
```



### Build the app for production
```bash
quasar build
```

### Customize the configuration
See [Configuring quasar.config.js](https://v2.quasar.dev/quasar-cli-vite/quasar-config-js).
